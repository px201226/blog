2021-02-25 > 알고리즘

# 문제

https://programmers.co.kr/learn/courses/30/lessons/72413 (opens new window)

# 풀이

최단 거리 알고리즘을 사용하는 문제로 일반적으로 출발 정점 u 에서 v 까지 가는 문제가 많은데, 이 문제에 경우에는 두 사람이 정점 u 에서 임의의 중간 정점 k를 지나 각각 v,w 지점까지의 비용이 최소인 경우를 구해야 한다.

출발 정점 u와 도착 정점 v,w를 알려주므로, 식으로 표현한다면,

minCost = cost(u,k) + cost(k,v) + cost(k,w)

minCost가 최소가 되는 정점 k를 구해야 한다. 정점 k는 정점의 수가 200개므로 완전탐색을 통해 구할 수 있으며, 비용을 구하는 cost 함수는 다익스트라 알고리즘이나 플로이드 알고리즘을 사용하여 구할 수 있다.

여기서는, cost 함수 호출 중복을 막기 위해, 플로이드 알고리즘을 통해 구현한다.

# 코드


/*
 * https://programmers.co.kr/learn/courses/30/lessons/72413
  Floyd 알고리즘을 이용해 A,B 탑승자의 K 지점을 구해
  * cost(s,k) + cost(k,a) + cost(k,b)가 최소가 되는 점을 구한다.
 * */

class Solution {
    public int solution(int n, int s, int a, int b, int[][] fares) {
        int answer = Integer.MAX_VALUE;

        int[][] cost = new int[n+1][n+1];
        for(int[] fare : fares){
            cost[fare[0]][fare[1]] = fare[2];
            cost[fare[1]][fare[0]] = fare[2];
        }

        for(int k=1; k<n+1; k++){
            for(int i=1; i<n+1; i++){
                for(int j=1; j<n+1; j++){
                    if(i==j) continue;
                    if(cost[i][k] == 0 || cost[k][j] == 0) continue;
                    if(cost[i][j] == 0)
                        cost[i][j] = cost[i][k] + cost[k][j];
                    else
                        cost[i][j] = Math.min(cost[i][j], cost[i][k] + cost[k][j]);
                }
            }
        }

        for(int k=1; k<n+1; k++){
            if(cost[s][k] + cost[k][a] + cost[k][b] == 0) continue;
            answer = Math.min(answer, cost[s][k] + cost[k][a] + cost[k][b] );
        }
        return answer;
    }
}
